# Bastion — marketing site

The marketing site for **Bastion, the AI Trust Router**: a self-hosted layer
between your applications and every AI model that classifies each prompt by data
sensitivity, keeps regulated data inside your boundary, and routes everything
else to the cheapest model that clears the quality bar — with a tamper-evident
audit of every decision.

A single static `index.html` (no build step), published with **GitHub Pages**.

## Local preview

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```
